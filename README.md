## Algorithms

### 1. TD3+BC (Baseline)
Fujimoto & Gu, 2021. The simplest offline RL method that works.
Adds a behavior cloning term to the TD3 actor loss:

```
L_actor = -О»В·Q(s, ПЂ(s)) + ||ПЂ(s) - a_data||ВІ
О» = О± / mean|Q(s,a)|   в†ђ lambda normalization
```

### 2. CQL вЂ” Conservative Q-Learning
Kumar et al., NeurIPS 2020. Adds a penalty that **lower-bounds** Q on OOD actions:

```
L_CQL = L_Bellman + О± В· (logsumexp_a Q(s,a) - E_data[Q(s,a)])
```
The logsumexp term pushes Q **down** on all actions; the second term pushes Q **up** on data actions. Net result: Q is provably conservative.

### 3. IQL вЂ” Implicit Q-Learning
Kostrikov et al., ICLR 2022. Avoids OOD actions **entirely** via expectile regression:

```
# V-network: approximate high quantile of Q
L_V = E[|П„ - I(Q-V < 0)| В· (Q-V)ВІ]    # expectile loss

# Q-network: Bellman with V(s') bootstrap (no max_a Q!)

# Actor: advantage-weighted behavior cloning

```

## Quick start (CPU only)

```bash
# Install
pip install -r requirements.txt

# Run CQL on synthetic LQR data (no download needed, ~10 min on CPU)
python train.py --agent cql --dataset hopper-synthetic --steps 100000

# Run IQL
python train.py --agent iql --dataset hopper-synthetic --steps 100000 --expectile 0.7 --temperature 3.0

# Run TD3+BC baseline
python train.py --agent td3bc --dataset hopper-synthetic --steps 100000

# With real Minari dataset (downloads ~50MB)
python train.py --agent cql --dataset D4RL/hopper/medium/v2 --steps 300000
```
